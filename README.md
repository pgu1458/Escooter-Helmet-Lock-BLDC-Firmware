## 프로젝트 소개

공유 전동킥보드 이용 시 발생할 수 있는 사고를 사전에 감지·차단하기 위해, 아두이노 기반으로 **4가지 안전 기능을 통합한 시스템**을 4인 팀으로 개발하고 한국전자통신학회 논문지에 게재한 프로젝트입니다.

전체 시스템은 ① 헬멧 착용 감지 + BLDC 제어 ② 방향지시등 ③ 다중 탑승 방지 ④ 헬멧 보관함 — 4개 모듈로 구성되며, App Inventor 기반 모바일 앱으로 실시간 제어가 가능합니다. 저는 그중 **헬멧 착용 감지·BLDC 모터 제어·헬멧 보관함·모바일 앱**을 담당했습니다.

## 주요 기능

### 🪖 1. 헬멧 착용 감지 → BLDC 모터 제어

전동킥보드의 안전 인터록 기능입니다. 헬멧 안쪽에 부착한 **터치센서로 착용 여부를 판별**하고, 미착용 상태에서는 BLDC 모터의 동작을 차단해 주행 자체가 불가능하도록 했습니다.

| 조건 | 동작 |
|------|------|
| 헬멧 착용(터치센서 ON) | BLDC 잠금 해제 → 주행 가능 |
| 헬멧 미착용(터치센서 OFF) | BLDC 잠금 → 주행 차단 |

**실험 결과 (논문 보고):** 헬멧 감지·모터 제어 모두 **100% 정확도**.

단순한 ON/OFF 스위치가 아니라 헬멧 안쪽의 자연스러운 접촉을 감지하기 위해 터치센서 위치와 감도를 여러 차례 조정했습니다. 헬멧을 벗으면 즉시 모터가 잠기고, 다시 쓰면 잠금이 풀리는 흐름을 끊김 없이 동작하도록 구현했습니다.

### 🔐 2. 헬멧 보관함 — RFID + 홀센서 이중 검증

헬멧 대여·반납을 자동화하는 무인 보관함입니다. 단순히 RFID만 쓰면 헬멧이 실제로 들어왔는지 아닌지 확인이 안 되므로, **RFID + 홀센서 이중 검증** 구조로 설계해 반납 검증 정확도를 100%로 만들었습니다.

| 단계 | 사용 센서 | 검증 내용 |
|------|----------|----------|
| 1차 | RFID 리더 | 헬멧에 부착된 태그 UID 인식 |
| 2차 | 홀 센서 | 헬멧이 보관함 내부에 실제로 안착했는지 자기장 감지 |

**대여 흐름**
1. 사용자가 보관함 앞에서 인증
2. 잠금 해제 → 헬멧 꺼냄
3. 보관함 닫힘

**반납 흐름**
1. 헬멧을 보관함에 넣음
2. RFID 태그 인식 + 홀센서로 안착 확인 (둘 다 OK여야 통과)
3. 반납 완료 처리 → 보관함 잠금

**실험 결과 (논문 보고):** 헬멧 보관함 **100% 반납 검증 성능**.

RFID 단독 검증이라면 태그만 가까이 대도 반납으로 인식되는 허점이 있는데, 홀센서로 물리적 안착을 한 번 더 확인하기 때문에 이런 우회가 막힙니다.

### 📱 3. App Inventor 모바일 앱

App Inventor 기반 안드로이드 앱으로, 블루투스를 통해 킥보드와 통신합니다. 헬멧 보관함 잠금 해제·상태 조회·주행 제어 등을 사용자가 직접 앱에서 트리거할 수 있습니다.

**통신 사양 (논문 보고):**
- 프로토콜: 블루투스 (HC-05/06 클래식 BT)
- 안정 통신 거리: **15m 이내**
- 응답: 실시간 제어 가능
